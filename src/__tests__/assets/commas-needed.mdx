# Commas Are Needed for This Post

This post needs commas between citations, so make sure your hippocampus is working (important in episodic memory[^hippocampus-review-1][^hippocampus-review-2][^hippocampus-review-3][^hippocampus-review-4]), since you'll want to remember this article (maybe).

So, as it turns out, the hippocampus probably differs between primates and rodents, as studies in primates couldn't replicate the range of neuronal selectivity found rats and mice[^diff-1][^diff-2], but why? One study decided to investigate this and learned that it boils down to a fundamental difference in how these animals view (and mentally represent) the world[^main-paper].
